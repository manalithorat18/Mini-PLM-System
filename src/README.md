# Mini PLM System

A simplified Product Lifecycle Management (PLM) system developed in C++ to manage engineering components, dependencies, and version tracking.

## Features

- Add engineering components
- Display component information
- Modular architecture using OOP
- STL vector-based storage
- Command-line interface
- Git/GitHub integration

## Technologies Used

- C++
- STL
- Object-Oriented Programming
- Git & GitHub
